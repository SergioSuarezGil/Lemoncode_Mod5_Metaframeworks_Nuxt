# Desafíos implementados

- Listado de casas en /list
- Detalle de casa en /houses/:id
- Redirección de / a /list
- Buscador por nombre y ubicación (sin tildes)
- Grid responsive de cards
- Card clicable hacia detalle
- Cabecera con identidad CR
- Imágenes servidas desde la API local
- Estilos con Tailwind CSS (paleta azul/verde, layouts con Flexbox)
- Botón visual de "Reservar" en el detalle (solo interfaz)
- A mayores he creado un HouseReviewSchema porque lo hacemos mucho en mi empresa para mapear la resta de la API.
- Y tambien he añadido un script que arranca api y front en paralelo para facilitar la ejecución del proyecto.
